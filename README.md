## Security & `.gitignore` Details
To maintain the security of this project, a `.gitignore` file is utilized. This ensures that sensitive information and unnecessary files are not tracked by version control or exposed publicly.

The following items are ignored:
- `.env`: Contains private environment variables, API keys, and database credentials. It is extremely important that this file is kept secret.
- `__pycache__/` and `*.pyc`: Compiled python files generated locally, varying across environments.
- `file_versions/` and `upload_bots/`: Local server directories that handle temporary caching and user-uploaded data, respectively.
